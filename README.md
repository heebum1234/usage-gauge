# Usage Gauge

Claude Code와 Codex의 잔여 사용량을 실시간으로 보여주는 데스크탑 floating widget.

## Features

- **Floating Widget** - 항상 위에 떠 있는 퍼센트 게이지 UI
- **사용량 모니터링** - Claude Code (`/usage`), Codex (`/status`) 잔여량 표시
- **알림** - 10% 단위로 시스템 알림 (toast/notification)

## Tech Stack

- **Framework**: Electron
- **Target**: Windows (primary), macOS (secondary)
